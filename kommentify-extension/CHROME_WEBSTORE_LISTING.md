## ğŸ“‹ PERMISSION JUSTIFICATIONS

When submitting, you'll need to justify each permission:

| Permission | Justification |
|------------|---------------|
| `storage` | Store user preferences, settings, and automation data locally |
| `unlimitedStorage` | Store scheduled posts, connection history, and analytics data |
| `tabs` | Open LinkedIn tabs for automation and detect active LinkedIn sessions |
| `scripting` | Inject content scripts to interact with LinkedIn pages |
| `activeTab` | Access the current LinkedIn tab to perform automation actions |
| `alarms` | Schedule automation tasks, reminders, and post scheduling |
| `notifications` | Alert users about automation status, scheduled posts, and important updates |

### Host Permission Justifications:

| Host | Justification |
|------|---------------|
| `linkedin.com` | Core functionality - automation runs on LinkedIn |
| `api.openai.com` | AI comment generation using OpenAI API |
| `*.vercel.app` | Backend API for user authentication and data sync |
