Take a screenshot of each tab in the Chrome extension
Save as screenshot.png in the corresponding folder:

public/extension-screenshots/writer/screenshot.png
etc.
Refresh the page - screenshots will be used automatically
Running the Converter
bash
cd backend-api
node scripts/convert-extension-tabs.js
The dev server is running at http://localhost:3001. You can open the browser preview to see the current state. Once you add screenshots to the folders, they will automatically appear in the live preview.
